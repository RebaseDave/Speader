# Speader

A personal RSVP reading app for Android, built with Flutter.

## Features

- **RSVP mode** — word-by-word display with configurable WPM, pauses, and ORP highlighting
- **Paragraph mode** — full paragraph display with tap navigation
- **Companion system** — 11 collectible companions that level up as you read
- **Reading statistics** — WPM history by mode, streaks, weekly goals, and session tracking
- **EPUB & TXT support** — with chapter navigation and token caching
- **Customizable** — fonts, color profiles, ORP settings, spotlight, and more

## Tech

Flutter · Dart · Riverpod · SQLite (sqflite)

## Notes

Personal project, not intended for public distribution.
